# Análisis de Big Data: Casos de COVID-19 en Argentina (2019-2025)

Este proyecto fue desarrollado como defensa final para la obtención de mi título como **Técnico en Programación**. Consiste en un análisis exhaustivo de la evolución de la pandemia de COVID-19 en la República Argentina, procesando datos para identificar patrones de contagio y efectividad de medidas sanitarias.

## 📊 Descripción del Proyecto
El análisis abarca desde los inicios de la pandemia en 2019 hasta las proyecciones y datos recolectados hacia 2025. Se utilizaron técnicas de Big Data para la limpieza, procesamiento y visualización de grandes volúmenes de información pública.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python
* **Entorno:** Google Colab
* **Librerías principales:** Pandas, Matplotlib, Seaborn, NumPy.

## 📂 Contenido del Repositorio
* `analisis_covid.ipynb`: Notebook con el flujo de trabajo técnico (limpieza de datos y gráficos).
* `informe_conclusiones.pdf`: Documento detallado con el ensayo y las conclusiones finales de la defensa.

## 🔗 Fuente de Datos
Los datos utilizados en este análisis provienen del portal oficial:
* [Buenos Aires Data - Casos COVID-19](https://datos.gob.ar/dataset/salud-covid-19-casos-registrados-republica-argentina)
